# 规范

1. 每个标题之间空一行。

2. 关于引用：指明书名与对应页数。

3. 每一篇必标注最后修改日期。

4. 例子之前要用分割线。

5. 图片的命名：同级子文件夹、命名细化到知识点，驼峰命名法。

6. 图片的存储：图片太多，准备使用二级目录。

7. 关于开头：
   $$
   name \ \ \ ...(10 * spaces) \  \ date \ \ \ version+description
   $$
